# UE 6.3.2 PWA — Activité 1

**Page web de réservation d'hôtel — HTML5 + CSS3 responsive**

Auteur : Mikail Lekesiz · LP DWCA 2025/2026 · Université de Strasbourg

## Énoncé

Créer une page web en HTML5/CSS3 permettant d'effectuer des réservations dans un hôtel.

Contenu requis :
- À gauche : une image de découverte de l'hôtel
- À droite : un formulaire de réservation avec les champs suivants :
  - Nom et prénom
  - Nombre d'adultes
  - Nombre d'enfants
  - Date d'arrivée
  - Date de départ
  - Options (aucune option, repas compris, service de location véhicule)
  - Bouton de soumission
- Adaptation responsive : smartphone, tablette, PC
- Hébergement HTTPS sur AlwaysData

## Stack

- HTML5 sémantique (header / main / section / form / footer)
- CSS3 (variables, grid, flexbox, media queries, mobile-first)
- JavaScript vanilla (validation + UX confirmation)
- Aucune dépendance externe (sauf image hébergée sur Unsplash CDN)
